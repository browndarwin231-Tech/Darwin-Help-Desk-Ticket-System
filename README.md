# Darwin Help Desk Ticket System

A Python-based Help Desk Ticket System that allows users to create, track, search, and close support tickets through an interactive command-line interface.

## Features

- Create new support tickets
- Automatically assign ticket IDs
- View all open and closed tickets
- Search for tickets by ID
- Close existing tickets
- Interactive menu system
- Beginner-friendly Python code

## How It Works

The application simulates a real Help Desk environment by allowing users to submit support requests and manage ticket statuses.

### Available Options

1. Create Ticket
2. View Tickets
3. Search Ticket
4. Close Ticket
5. Exit

## Example

```text
=== Darwin Help Desk Ticket System ===

1. Create Ticket
2. View Tickets
3. Search Ticket
4. Close Ticket
5. Exit

Choose an option: 1

Employee Name: John Smith
Issue Description: Unable to access email

Ticket #1000 Created
```
